    Function createDatabase

    • createDatabase(config: DatabaseConfig): Promise<IDatabase>

      Creates a new database instance with automatic provider detection.

      This is the main entry point for creating database connections. The function automatically detects browser capabilities and selects the best available storage provider unless explicitly specified.

      Parameters

      • config: DatabaseConfig

        Database configuration options

        Configuration options for creating a database instance.

        • name: string

          Unique database name per origin. Must be a valid identifier (alphanumeric, hyphens, underscores).

        • Optionalversion?: number

          Database schema version for handling migrations. Increment when making breaking changes to data structure.

          Default Value

          1
        • provider: DatabaseProviderType

          Storage provider selection.

          • 'auto' - Automatically detect best available provider
          • 'sqlite' - Force SQLite provider (WASM in Firefox, OPFS in Chrome)
          • 'json' - Force JSON provider with IndexedDB storage
        • Optionalbackend?: StorageBackendType

          Force specific storage backend (overrides auto-detection)

        • Optionaloptions?: {
              wasmUrl?: string;
              autoSave?: boolean;
              saveInterval?: number;
              objectStoreNames?: string[];
              area?: "local" | "sync" | "managed";
          }

          Provider-specific configuration options

          • OptionalwasmUrl?: string

            URL to SQLite WASM file

          • OptionalautoSave?: boolean

            Automatically save changes to storage

          • OptionalsaveInterval?: number

            Interval between auto-saves (milliseconds)

          • OptionalobjectStoreNames?: string[]

            Object store names to create

          • Optionalarea?: "local" | "sync" | "managed"

            Storage area to use ('local', 'sync', 'managed')

      Returns Promise<IDatabase>

      Promise that resolves to an IDatabase instance

      Throws

      When provider initialization fails

      Throws

      When database name is invalid

      Throws

      When unsupported provider is specified

      Example

      // Auto-detect best provider
      const db = await createDatabase({
        name: 'my-extension-db',
        provider: 'auto',
        version: 1
      });

      // Force specific provider
      const sqliteDb = await createDatabase({
        name: 'advanced-db',
        provider: 'sqlite',
        version: 2,
        options: { enableWAL: true }
      });

      // JSON provider with custom options
      const jsonDb = await createDatabase({
        name: 'simple-db',
        provider: 'json',
        options: { maxStorageSize: 100 * 1024 * 1024 } // 100MB
      });
